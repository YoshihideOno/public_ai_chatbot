## 📝 設計書作成ガイドライン

1. **Markdown形式**: すべての設計書はMarkdown形式で記述
2. **日本語記述**: すべての設計書は日本語で記述
3. **図表**: アーキテクチャ図などはMermaid記法を使用
4. **更新日**: 各設計書の最終更新日を記載

---

## 🔄 設計書更新フロー

1. 要件変更時に該当設計書を更新
2. 実装変更時に設計書との整合性を確認
3. PR作成時に設計書更新を含める
4. 定期的に設計書の正確性を確認
